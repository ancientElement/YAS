# 📱 iPhone 视频/照片压缩工具

一个纯前端实现的 iPhone 视频和照片压缩 WebApp，无需安装，浏览器即可使用。

## ✨ 功能特点

- 📸 **图片压缩**：支持 JPG、PNG 格式，按比例缩小尺寸
- 🎬 **视频压缩**：支持 MP4、MOV 格式，重新编码压缩
- 🎚️ **三档压缩**：清晰(70%) / 标准(50%) / 模糊(30%)
- 👀 **即时预览**：选择文件后立即显示原始文件
- ⚖️ **对比展示**：左右并排对比压缩前后效果
- 📊 **节省统计**：显示压缩后节省的存储空间比例
- 💾 **一键下载**：压缩完成后直接下载到本地

## 🚀 快速开始

### 方式一：直接打开（最简单）
直接用浏览器打开 `index.html` 文件即可使用。

### 方式二：Node.js 服务器部署

1. **确保已安装 Node.js**

2. **启动服务器**
   ```bash
   node server.js
   ```

3. **访问应用**
   - 本机：http://localhost:3000
   - 局域网：http://你的IP:3000

## 📝 使用说明

1. **选择文件**：点击上传区域或拖拽文件到页面
2. **选择等级**：
   - **清晰**：保留 70% 尺寸，画质较好
   - **标准**：保留 50% 尺寸，平衡画质和大小
   - **模糊**：保留 30% 尺寸，文件最小
3. **开始压缩**：点击"开始压缩"按钮
4. **对比预览**：左右查看压缩效果
5. **下载文件**：点击"下载压缩文件"保存

## ⚠️ 注意事项

- 压缩过程在浏览器本地完成，文件不会上传到服务器
- 视频压缩使用 WebM 格式输出
- 图片压缩后输出 JPG 格式
- 大文件压缩可能需要较长时间，请耐心等待

## 🔧 技术栈

- **前端**：原生 HTML5 + CSS3 + JavaScript
- **压缩技术**：Canvas API + MediaRecorder API
- **服务器**：Node.js 原生 http 模块
