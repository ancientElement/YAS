# 📱 iPhone 视频/照片压缩工具

一个支持多种压缩方案的 iPhone 视频和照片压缩 WebApp，可在浏览器本地压缩或服务器端压缩。

## ✨ 功能特点

- 📸 **图片压缩**：支持 JPG、PNG 格式，按比例缩小尺寸
- 🎬 **视频压缩**：支持 MP4、MOV 格式
- 🎚️ **三档压缩**：清晰(70%) / 标准(50%) / 模糊(30%)
- 👀 **即时预览**：选择文件后立即显示原始文件
- ⚖️ **对比展示**：左右并排对比压缩前后效果
- 📊 **节省统计**：显示压缩后节省的存储空间比例
- 💾 **一键下载**：支持下载原文件和压缩文件

## 🚀 快速开始

### Python 服务器

```bash
# 基础版
python3 server.py

# FFmpeg 版（支持音频压缩）
apk add ffmpeg  # 安装 FFmpeg
python3 server_ffmpeg.py
```

## 📝 使用说明

1. **选择文件**：点击上传区域选择视频/照片
2. **选择等级**：
   - **清晰**：保留 70% 尺寸，画质较好
   - **标准**：保留 50% 尺寸，平衡画质和大小
   - **模糊**：保留 30% 尺寸，文件最小
3. **开始压缩**：点击"开始压缩"按钮
4. **对比预览**：左右查看压缩效果
5. **下载文件**：
   - 点击"下载原文件"保存原始文件
   - 点击"下载压缩文件"保存压缩后的文件

## 🔧 压缩方案对比

| 方案 | 音频 | 速度 | 流量 | 适用场景 |
|------|------|------|------|----------|
| 纯前端 (index.html) | ❌ 无 | 慢 | 无上传 | 快速预览、小文件 |
| Node.js (server.js) | ❌ 无 | 慢 | 无上传 | 局域网共享 |
| Python (server.py) | ❌ 无 | 慢 | 无上传 | iSH 环境 |
| FFmpeg (server_ffmpeg.py) | ✅ 有 | 中等 | 需上传 | 需要音频、高质量 |

## ⚠️ 注意事项

- 需要上传文件到后端服务器压缩，保留音频，质量更好
- 大文件压缩可能需要较长时间

## 📦 依赖安装

### 安装 FFmpeg
```bash

apk add ffmpeg
ffmpeg -version  # 验证安装
```
